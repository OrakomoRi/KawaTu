# KawaTu ☕

Prosty projekt strony internetowej do rekomendacji kawy w zależności od preferencji smakowych użytkownika.

## Funkcje
- Wybór między smakami: gorzka, kwaśna, słodka
- Wyświetlenie rekomendacji na podstawie wyboru

## Technologie
- HTML
- CSS
- JavaScript (vanilla)
